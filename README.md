# Gerência de Fretes

Projeto de estudos desenvolvido em **Java** com o objetivo de aprender e praticar o desenvolvimento de **extensões Java para o ERP Sankhya**.

A aplicação implementa funcionalidades relacionadas ao gerenciamento de fretes, utilizando a infraestrutura disponibilizada pela plataforma Sankhya para criação de módulos Java.

> **Projeto desenvolvido exclusivamente para fins de estudo e aprendizado.**

---

## Objetivos

- Estudar o desenvolvimento de extensões Java para o Sankhya;
- Praticar integração com a API e infraestrutura da plataforma;
- Aplicar boas práticas de desenvolvimento Java;
- Experimentar regras de negócio voltadas à gestão de fretes.

---

## Tecnologias utilizadas

- Java
- Maven
- Sankhya SDK / Módulo Java
- JDBC
- Banco de Dados Oracle
- XML
- Git

---

## Funcionalidades

- Cadastro e gerenciamento de informações de frete;
- Implementação de regras de negócio personalizadas;
- Integração com entidades do ERP Sankhya;
- Validações de dados;
- Persistência das informações.

---

## Aprendizados

Durante o desenvolvimento deste projeto foram explorados conceitos como:

- Estrutura de projetos Maven;

---

## Como executar

### Pré-requisitos

- Java 17+ (ou a versão utilizada no projeto)
- Maven
- Ambiente Sankhya configurado
- Banco de dados compatível

### Clonar o projeto

```bash
git clone https://github.com/willdias/gerenciafretes.git
```

### Compilar

```bash
mvn clean install
```
